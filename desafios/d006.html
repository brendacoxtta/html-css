Tags em HTML5

Em HTML5 você pode usar a tag de abreviações <abbr> inserindo o texto que so será informado quando o mouse passar por cima

Também pode fazer uma lista não ordenada ou ordenada, usando a tag <ul> ou <ol> e dentro inserir a ferramenta <li> indicando cada itém, use a ferramenta type dentro da u/ol para definir o marcador
type ul: disc, square e circle
type ol: 1, A, a, I, i
